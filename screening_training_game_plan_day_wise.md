# 🎮 Screening Training Game Plan (Day-wise)

---

# 📅 DAY 3: 🎓 Education, Graduation & Basic Eligibility

## 🎯 Objective:
Build strong foundation in academic eligibility and initial screening checks

## 🧠 Coverage (SOP Mapping):
- Academic stream / specialization
- CGPA validation
- Graduation year & date validation
- Degree type validation
- Computer Science rules
- University verification
- Initial eligibility verification

## 🎮 Games

### 1. Degree Sorter
- Categorize degrees as Accept / Reject

### 2. CS Rule Puzzle
- Apply CS eligibility logic

### 3. Graduation Gate
- Validate graduation range

### 4. Date Conflict Resolver
- Resolve resume vs application mismatch

### 5. University Validator
- Validate region and legitimacy

### 6. Campus Confusion
- Identify correct campus location

### 7. Community College Check
- Validate eligibility criteria

### 8. Flag It!
- Identify study abroad / exchange cases

### 9. Profile Basics Check (NEW)
- Validate:
  - CGPA
  - Academic stream
  - Graduation year
  - Location

### 10. Incomplete Education Case
- Decide: Proceed / Reject / Escalate

## 🎯 Final Game
### Education Audit Challenge
- 10 mixed candidate profiles
- Justify decisions

- Identify first active application

- Validate rehire eligibility

- Calculate total and managerial experience

  - Education responses

### 7. Shift Fit Game

### 9. Escalation Decision Game
- Choose Proceed / Reject / Escalate

### 11. Qualification Deep Dive (NEW)

  - Availability during studies
  - Employment history

## 🎯 Final Game
### Profile Investigation Room
- Multi-layer candidate scenarios
- Identify all issues before decision

# 🌟 MAIN GAME (DAY 5)

## 🎮 Screening Command Center

### 🧑‍💻 Simulation Flow:
1. Review candidate (resume + application)
2. Check system (duplicate + rehire)
3. Apply rules:
   - Education (Day 3)
   - Experience (Day 4)
   - Location & Visa (Day 5)
4. Make decision:
   - Proceed
   - Reject
   - Escalate
5. Provide justification

### 🎮 Mechanics:
- SLA timer
- Accuracy scoring
- Audit risk meter
- Difficulty scaling

### 🎯 Final Challenge:
Global Hiring Day
- 25–30 candidates
- Mixed regions (India, US, Canada)
- High complexity

By end of Day 5, agents will:
